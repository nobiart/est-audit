ТОО «Э.С.Т.аудит» является официальным учебным центром Института профессиональных управляющих Аскери (г. Москва), который предлагает образование по международному стандарту АССА на русском языке. Сегодня Институт предлагает две специализации:

  • для управляющих широкого профиля, экономистов, управленцев — официальный диплом по специализации «финансовое управление» (курс «Практическое управление финансами»);
  • для специалистов по МСФО (аудиторов, консультантов, финансовых аналитиков, профессиональных бухгалтеров, финансовых директоров) — официальный диплом по МСФО (курс «Международные стандарты финансовой отчетности»).

В дальнейшем вы сможете постоянно повышать квалификацию на специальных семинарах и посещая дополнительные курсы.

Более подробно с программой обучения и порядком сдачи экзаменов Вы можете ознакомиться, пройдя по ссылке: www.askeri.ru
